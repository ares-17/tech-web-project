## Deploy
Per eseguire i container:
```sh
docker-compose up
```
Per eseguire i container e utilizzare il qrcode, aggiungere come parametro l'indirizzo IP sul quale sono eseguiti i container Docker:
```sh
API_HOST=YOUR_LOCAL_IP docker-compose up
```
